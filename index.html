Project Overview Projektüberblick

Making core cosmology topics easier to explore. Zentrale kosmologische Themen verständlicher machen.

This repository will grow into a set of browser-based visualizations that explain key ideas about our universe in a clear and accessible way. Dieses Repository wächst zu einer Sammlung von browserbasierten Visualisierungen heran, die wichtige Ideen über unser Universum klar und zugänglich erklären.
